# PanicLens

PanicLens is a SaaS for iPhone panic-full diagnosis, with structured parsing, versioned rules, hypothesis resolution, calibration by model family, and operational workflows for bench technicians.

## Current engine

- Engine version: `3.0.0`
- Ruleset version: `3.0.0`
- Model calibration version: `1.0.0`

## What it does

- Parses raw panic-full logs into structured metadata and signals
- Resolves competing hypotheses with evidence and conflict tracking
- Prioritizes tests and repair actions for bench use
- Applies model-family calibration to improve diagnosis context

## Validation

The engine is covered by automated tests and the project builds successfully with the current `main` branch state.
